# Dotfiles

## Dependencies

### stow
```
brew install stow
```


## Copy dotfiles

Clone this repo

```git
git clone git@github.com:MilanDP123/dotfiles.git ~/dotfiles
```

## set up dotfiles

Go to the dotfiles dir and create the symlinks.
Make shure there are no clonflicting files/dirs in the root dir

```
cd dotfiles
stow .
```
